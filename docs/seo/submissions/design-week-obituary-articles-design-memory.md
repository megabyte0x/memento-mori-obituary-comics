## Core Ask

Pitch a column, opinion piece, or project story about obituary articles as designed public memory interfaces. The link should be contextual, not requested.

## Best Angle

Design as problem solving for end-of-life publishing: how page structure, captions, source trails, privacy decisions, images, and summaries help obituary articles serve families, editors, readers, classrooms, and search systems without turning a real life into a generic notice.
