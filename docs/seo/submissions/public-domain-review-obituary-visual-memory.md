# Public Domain Review Submission Package

Status: prepared, not sent.

Use only with a public-domain image/source list and an author who can write an original cultural-history essay.

## Source-Checked Route

- Submission page: `https://publicdomainreview.org/about/submissions/`
- Email: `submissions@publicdomainreview.org`
- Best FinalNotes target: `https://www.finalnotes.page/obituary-stories/`

## Core Ask

Pitch an essay about public-domain obituary images, mourning portraits, memorial cards, death notices, or related visual memory material. The FinalNotes link should be a contemporary context link only.

## Best Angle

Public-domain memorial imagery shows how death becomes public record: faces, names, captions, symbols, borders, and printed notices outliving the private grief that produced them. A source-backed visual obituary project can be referenced briefly as a modern continuation of the evidence-and-image problem.

## Materials To Prepare

- Public-domain source list.
- 300-500 word proposal or complete essay.
- Author bio.
- Recent clips.
- Image links or JPEG/PNG attachments if allowed.
- Rights notes for every image.

## Do Not Send Until

- The central works are public domain.
- The essay can stand without promotional framing.
- The author can demonstrate facility with the subject.
- Image rights and captions are ready.
