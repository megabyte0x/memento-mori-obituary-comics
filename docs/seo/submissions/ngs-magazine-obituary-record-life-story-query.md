## Why This Qualifies

The current magazine page says it publishes articles about genealogical records, collections, repositories, research techniques, databases, case studies, and genealogical problem-solving. Its author guidelines encourage prospective authors to query before submitting unsolicited articles, and the query should summarize the proposed article plus the author's experience. The best ask is a query for a technique-focused genealogy article. It should not be a personal essay, product pitch, or general obituary-writing article. A qualified author should be able to write in third person, cite sources with footnotes, and follow NGS/Chicago/Evidence Explained style.

## Query Subject

Query: Using obituary records to build sourced life-story narratives

## Query Email

Hi NGS Magazine editor,

I would like to query a possible article for NGS Magazine:

"Using Obituary Records to Build Sourced Life-Story Narratives"

The proposed article would be a technique-focused piece for family historians. It would show how to use obituary records as a starting point for a sourced life-story narrative while avoiding common errors: assuming every obituary detail is accurate, adding unsourced emotional context, losing track of citations, or turning a brief notice into unsupported biography.

Possible sections:

- What obituary records can and cannot prove.
- How to extract names, places, events, occupations, affiliations, and relationship clues.
- How to resolve conflicts between an obituary and other records.
- How to document each narrative claim.
- How to present a life-story summary while preserving the evidence path.

The article would not be a personal essay or family history. It would use compact examples to show a repeatable method that readers can apply in their own research.

Relevant background project:
https://www.finalnotes.page/how-to-write-an-obituary-story/

Best,
[sender name]
