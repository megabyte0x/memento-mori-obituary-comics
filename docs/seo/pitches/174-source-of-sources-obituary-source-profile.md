## Why This Fits

Source of Sources sends media queries from journalists to expert sources up to three times per day and explicitly warns sources to respond only when they have something relevant, timely, and accurate to add. This is a good channel for obituary-related journalist queries, but it requires discipline because off-topic replies are prohibited.

## Signup Positioning

Use Source of Sources as a broad media query stream for:

- obituary writing and obituary examples
- digital memorialization and funeral technology
- AI obituary generators and sensitive content
- family-history records and death notices
- obituary scams, copied obituaries, and privacy
- visual biography and memorial art

## Response Rule

Only answer when the query directly matches FinalNotes expertise. For every response:

- start with a direct answer
- include one concrete obituary or digital memorial example
- include disclosure
- include one context link only if useful
- offer follow-up availability

## Final Setup Checklist

- Subscribe with the real sender email.
- Filter incoming queries by the fit list above.
- Use the response bank for quote-ready replies.
- Log every sent response and any live mention.
- Do not reply to unrelated query digests.
