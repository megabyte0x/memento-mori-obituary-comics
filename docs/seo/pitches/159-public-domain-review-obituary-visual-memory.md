I would like to propose an essay on public-domain obituary images, mourning portraits, memorial cards, and illustrated death notices as forms of visual memory.

The essay would follow how death records and memorial images turned private grief into public evidence: a name, face, caption, object, costume, border, or printed image that later becomes source material for historians, artists, genealogists, and family memory.

The contemporary context would be brief. Projects such as Memento Mori Obituary Comics show how visual obituary storytelling still depends on source trails, captions, rights, and evidence:
