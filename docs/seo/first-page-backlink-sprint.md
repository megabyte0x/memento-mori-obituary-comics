## Selection Criteria

Use these criteria for the first outreach wave:

- Topical fit with obituary stories, obituary journalism, visual biography, grief comics, genealogy, classrooms, or funeral-planning resources.
- Clear editorial, contact, submission, or resource-suggestion path.
- Natural anchor fit without exact-match pressure.
- A relevant FinalNotes target page that already answers the recipient's audience need.
- No paid link insertion, link exchange, generic directory, or automated guest-post network.
